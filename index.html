Topic: Chainless Bicycle System

Chainless Bicycle System (CBS) is a setup which makes bicycles run on the road without chains. CBS uses a shaft-driven concept; it uses a drive-shaft for the transmission of power from the pedals to the wheels in place of chains. In the present era, development in internal gear technology produces various advantages. So, I decided to construct a bicycle using the shaft-driven system rather than using chain-driven.
